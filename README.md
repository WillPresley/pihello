## Installation

### Requirements

- A running [pi-hole](https://pi-hole.net/) instance
- [Python 3](https://www.python.org/downloads/)
- [GNU make](https://www.gnu.org/software/make/)

### Install

```bash
git clone https://github.com/pavelgar/pihello.git
cd pihello
make install
```

### Usage

**Basic**

```bash
pihello pihole.localdomain
```
